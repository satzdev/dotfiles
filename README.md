# Satz's dotfiles


## Overview of files

#### Bash
* `.bash_profile`
* `.bashrc`
* `.bash/*.sh` - os specific settings

#### Sublime text 3
* `*.sublime-settings`

#### Etc.
* VIM - `.vimrc`
* tmux - `.tmux.conf`

## Installation

```bash
git clone https://github.com/satzdev/dotfiles.git
```

Copy files into specific directories.
